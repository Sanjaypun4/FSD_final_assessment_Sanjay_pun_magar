PROJECT DESCRIPTION:-
The Student Management System is a web-based application developed using PHP and MySQL.
This project is designed to manage student-related information in an organized and efficient way.

It helps the institution to handle:

Student details
Staff details
Courses
Attendance
Grades
Fees
User login and roles

Instead of maintaining records manually, this system stores all data digitally, making it easy to access, update, and manage.

KEY FEATURES:-

Secure login system with roles
Student information management
Staff management
Course management
Attendance tracking
Grade and result management
Fee payment tracking
User-friendly interface
Database-driven system using MySQL

Admin (Super Admin)
The Admin has full control over students and staffs

Admin can:
Manage all users (add, edit, delete)
Add and manage students and staff
Assign roles to users


Professor / Teacher
The Professor is responsible for academic activities.

Professor can:
View assigned students
Mark attendance
Enter and update grades
Add marks and results
View course-related student performance

Finance / Accountant

The Finance (Accountant) role handles all payment-related work.

Finance can:

Manage student fees
Update payment status (paid/unpaid)
Record paid amount
View fee reports


Student
The Student can access their own academic information.

Student can:
View personal profile
Check attendance
View grades and results



Staff (Non-Teaching Staff)
The Staff role supports administrative tasks.
Staff can:
Manage student records
Assist in course management
View reports as assigned
Support daily administrative operations
